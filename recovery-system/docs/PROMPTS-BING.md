# PROMPTS PARA BING IMAGE CREATOR
## https://www.bing.com/images/create

### CÓMO USARLO:
1. Ve a https://www.bing.com/images/create
2. Inicia sesión con tu cuenta Microsoft (Outlook, Hotmail, etc.)
3. Copia el prompt de abajo
4. Pégalo en el cuadro de texto
5. Haz clic en "Create"
6. Descarga la imagen que más te guste (clic derecho → Guardar imagen)
7. Guarda como: `nombre-producto-tipo.jpg` en `recovery-system/public/images/`
